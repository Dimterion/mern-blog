## <a id="features"></a>📌 Features

- Functionality to add and edit comments.
- Functionality to monitor various statistics for the site.

## <a id="tech-stack"></a>⚙️ Tech Stack

- React
- React Router
- Redux
- Tailwind CSS
- Flowbite React
- Quill (initially used ReactQuill)
- Mongo DB
- Express
- Node.js
